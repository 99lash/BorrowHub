BORROWHUB

Users
- MIS/CSD Staff
- MIS/CSD Admin

Features of MIS/CSD Staff
- Home
- Inventory (View only)
- Transaction (Borrow/Return)
- Logs (Transactions/Audits)
- Account Settings

Features of MIS/CSD Admin
- Home
- Inventory (Can Update, Delete Inventory Items)
- User Management (Can Update, Delete, Reset user password to default).
- Transaction (Borrow/Return)
- Logs (Transactions/Audits)
- Account Settings
